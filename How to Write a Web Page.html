How to Write a Web Page

Isaiah Comer

In this article, I wish to instruct you on how to make a shareable web article in a very short amount of time. I could just tell you how to write a Word document or download the Wordpress app, but custom web pages are so much faster and more shareable.

Do you want to be able to code on your mobile device? If so, go ahead and download Git2Go from the App Store on your iPhone, iPad, or iPod.

Our first step is simple: Write down all of the words you want on your web article. I don't care what you use to write, just make sure you write in plain text (no markdown for this template).

Next, download this template to your computer or phone. It ends in .html, so to make sure you can open and edit it you need to download an editor like iA Writer, Ulysses, or Atom (Git2Go has an editor built in).

Next step: Create a Github account. Just go to github.com, and sign up. Then create a new repository, and make sure that the repository's name is yourusername.github.io. Insert your Github user name for yourusername.

Next, you need to copy and paste your article in between the `` tags. Follow the instructions to separate the 'body' of your writing into paragraphs.
